# CSS
CSS — це мова, яку ми використовуємо для оформлення HTML-документа.
CSS описує, як мають відображатися елементи HTML.<br />
# STRUCTURE CSS
**CSS Example**  <br />
```javascript
body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}
```
